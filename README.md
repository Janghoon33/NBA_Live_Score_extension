# 🏀 NBA Live Score

**코딩하면서 NBA 실시간 점수와 중계를 VS Code 안에서 바로 확인하세요.**

브라우저를 열 필요 없이 Explorer 패널에서 오늘의 경기 일정, 실시간 점수, 플레이별 중계까지 한눈에 볼 수 있습니다.

---

## 주요 기능

- **오늘의 경기 일정** — VS Code를 열면 오늘 예정된 NBA 경기 목록을 바로 확인
- **실시간 점수** — 경기 중 20초마다 점수 자동 갱신 (전체 경기 업데이트)
- **경기 선택** — ★ 버튼으로 원하는 경기만 골라서 중계 피드 활성화 (플레이-바이-플레이 & 기록지)
- **플레이별 중계 피드** — 선택한 경기의 모든 액션을 **6초 간격**으로 폴링하여 거의 실시간으로 표시 (팀 로고 포함):
  - 🤟 3점슛 성공 / ❌ 3점슛 실패
  - 💥 덩크 / 🏀 2점슛 / 🤸 앨리웁
  - ☝️ 자유투 성공 / ❌ 실패
  - 💪 공격 리바운드 / 🫳 수비 리바운드
  - ✊ 스틸 / 🚫 블록
  - 🔄 턴오버 (패스 실수, 오펜시브 차지, 트래블링, 아웃오브바운즈 등) / ✋ 파울
  - ⚡ 점프볼 (양 선수명 표시, Tip to 정보 포함) / ⏸️ 타임아웃 / 🔔 쿼터 종료 등
- **팀 로고로 구분** — 각 플레이 행에 팀 로고가 표시되어 어느 팀의 액션인지 즉시 확인 (점프볼은 팀 표시 없음)
- **정확한 선수 팀 매핑** — 선수 ID 기반으로 팀을 확인하여 파울/턴오버 시에도 정확한 팀 표시
- **새 플레이 하이라이트** — 새로운 플레이 도착 시 슬라이드 인 애니메이션으로 즉시 눈에 띔
- **스마트 자동 갱신** — 하프타임, 쿼터 사이에도 계속 갱신 (경기 종료/예정 상태만 중지)
- **스크롤 위치 보존** — 기록지 보는 중에도 자동 갱신 후 스크롤 위치 유지
- **한국어 / 영어 지원** — 헤더의 KO / EN 버튼으로 즉시 전환
- **선택 기억** — 선택한 경기는 VS Code를 재시작해도 유지 (날짜가 바뀌면 자동 초기화)
- **기록지** — 선택한 경기의 선수별 기록(득점, 어시, 리바운드, 스틸, 블록, FG%, 출전시간 등) 확인 가능. 선발 순으로 정렬되며 20초마다 자동 갱신

## 사용 방법

1. Explorer 패널 열기 (`Ctrl+Shift+E`)
2. 아래로 스크롤하여 **NBA LIVE SCORE** 섹션 찾기
3. 오늘 경기 목록이 자동으로 표시됨
4. 보고 싶은 경기 옆 **☆** 클릭 → **★** 로 바뀌며 중계 피드 & 플레이-바이-플레이 활성화
5. 경기 카드를 클릭하면 **기록지(Stats)** 패널 열기/닫기 가능
6. **↻** 버튼으로 수동 새로고침 가능
7. **KO/EN** 버튼으로 언어 전환 (한국어 ↔ 영어)

## 요구사항

- VS Code `1.74.0` 이상
- 인터넷 연결 (실시간 데이터 수신에 필요)

---

## 알려진 제한사항

- 플레이별 중계는 **경기 종료 및 경기 예정 상태**에서는 표시되지 않습니다 (진행/하프타임/쿼터 사이 중에는 갱신됨)
- 점수는 **20초 간격**, 플레이는 **6초 간격**으로 갱신됩니다
- 실제 업데이트 지연은 ESPN API 서버의 데이터 반영 속도에 따라 달라집니다
- **오늘 날짜 경기**만 표시됩니다 (미국 동부 시간 기준)
- 팀 리바운드는 선수 기록에 반영되지 않아 표시되지 않습니다
