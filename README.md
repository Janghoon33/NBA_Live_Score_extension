# 🏀 NBA Live Score

**코딩하면서 NBA 실시간 점수와 중계를 VS Code 안에서 바로 확인하세요.**

브라우저를 열 필요 없이 Explorer 패널에서 오늘의 경기 일정, 실시간 점수, 플레이별 중계까지 한눈에 볼 수 있습니다.

---

## 주요 기능

- **오늘의 경기 일정** — VS Code를 열면 오늘 예정된 NBA 경기 목록을 바로 확인
- **실시간 점수** — 경기 중 20초마다 점수 자동 갱신 (전체 경기 업데이트)
- **경기 선택** — ★ 버튼으로 원하는 경기만 골라서 중계 피드 활성화
- **플레이별 중계 피드** — 선택한 경기의 모든 액션을 **6초 간격**으로 폴링하여 거의 실시간으로 표시:
  - 🤟 3점슛 성공 / ❌ 3점슛 실패
  - 💥 덩크 / 🏀 2점슛 / 🤸 앨리웁
  - ☝️ 자유투 성공 / ❌ 실패
  - 💪 공격 리바운드 / 🫳 수비 리바운드
  - ✊ 스틸 / 🚫 블록
  - 🔄 턴오버 / ✋ 파울
  - ⏸️ 타임아웃 / 🔔 쿼터 종료 등
- **팀 로고로 구분** — 각 플레이 행에 팀 로고가 표시되어 어느 팀의 액션인지 즉시 확인
- **새 플레이 하이라이트** — 새로운 플레이 도착 시 슬라이드 인 애니메이션으로 즉시 눈에 띔
- **스마트 자동 갱신** — 선택된 라이브 경기가 있을 때만 6초 타이머 작동, 없으면 자동 중지
- **한국어 / 영어 지원** — 헤더의 KO / EN 버튼으로 즉시 전환
- **선택 기억** — 선택한 경기는 VS Code를 재시작해도 유지 (날짜가 바뀌면 자동 초기화)

---

## 사용 방법

1. Explorer 패널 열기 (`Ctrl+Shift+E`)
2. 아래로 스크롤하여 **NBA LIVE SCORE** 섹션 찾기
3. 오늘 경기 목록이 자동으로 표시됨
4. 보고 싶은 경기 옆 **☆** 클릭 → **★** 로 바뀌며 중계 피드 활성화
5. **↻** 버튼으로 수동 새로고침 가능

## 설치 방법

### VS Code Marketplace에서 설치
Extensions 패널(`Ctrl+Shift+X`)에서 **"NBA Live Score"** 검색 후 설치

### VSIX 파일로 직접 설치
```bash
code --install-extension nba-live-score-*.vsix
```

## 요구사항

- VS Code `1.74.0` 이상
- 인터넷 연결 (실시간 데이터 수신에 필요)

---

## 알려진 제한사항

- 플레이별 중계는 **진행 중인 경기**에서만 표시됩니다
- 점수는 **20초 간격**, 플레이는 **6초 간격**으로 갱신됩니다
- 실제 업데이트 지연은 ESPN API 서버의 데이터 반영 속도에 따라 달라집니다
- **오늘 날짜 경기**만 표시됩니다

> 본 확장 프로그램은 개인적, 비상업적 용도로만 제작되었습니다.
> ESPN은 언제든지 데이터 접근 방식을 변경하거나 제한할 수 있으며, 이 경우 기능에 영향이 있을 수 있습니다.
